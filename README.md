# Discord Auto

Автоматизированный инструмент для массовой рассылки сообщений в Discord через удобный графический интерфейс.

## Основные возможности

- 📨 **Рассылка по каналам** - массовая отправка сообщений в Discord каналы
- 💬 **Рассылка по ЛС** - автоматическая рассылка в личные сообщения
- 🔄 **Поддержка прокси** - работа через HTTP/SOCKS4/SOCKS5 прокси
- 📊 **Мониторинг токенов** - проверка валидности и статуса аккаунтов
- 📎 **Вложения** - отправка сообщений с файлами
- ⚡ **Многопоточность** - настраиваемое количество потоков для быстрой рассылки
- 📈 **Статистика** - детальная информация о результатах рассылки

## Требования

- Python 3.8+
- Windows/Linux/macOS

## Установка

1. Клонируйте репозиторий или скачайте архив:
```bash
git clone https://github.com/y1ki/Discord-Distribution
cd DiscordAuto
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Зависимости

- `PyQt6` - графический интерфейс
- `aiohttp` - асинхронные HTTP запросы
- `aiohttp-socks` - поддержка SOCKS прокси
- `aiofiles` - асинхронная работа с файлами
- `av` - обработка видео
- `numpy` - математические операции

## Запуск

```bash
python main.py
```

## Использование

### 1. Добавление аккаунтов

Добавьте Discord токены в раздел "Аккаунты". Формат:
```
token1
token2
token3
```

### 2. Добавление прокси (опционально)

Добавьте прокси в соответствующий раздел. Поддерживаемые форматы:
```
ip:port
ip:port:login:password
protocol://ip:port
protocol://login:password@ip:port
```

### 3. Настройка рассылки

1. Перейдите в раздел "Действия"
2. Выберите тип действия:
   - **Рассылка по каналам** - отправка в серверные каналы
   - **Рассылка по ЛС** - отправка в личные сообщения
3. Нажмите "Выбрать аккаунты"
4. Составьте сообщение и добавьте вложения (при необходимости)
5. Нажмите "Запуск"

### 4. Настройки рассылки

- **Количество потоков** - максимальное количество одновременных операций
- **Задержка** - пауза между отправками (секунды)

## Особенности

### Рассылка по каналам
- Автоматическое получение списка серверов и каналов
- Фильтрация каналов с правами на отправку
- Отслеживание уже отправленных сообщений

### Рассылка по ЛС
- Автоматическое получение списка DM каналов
- Поддержка личных и групповых чатов
- Защита от повторной отправки

### Проверка токенов
- Валидация токенов через Discord API
- Определение мертвых и заблокированных аккаунтов
- Проверка прав доступа
